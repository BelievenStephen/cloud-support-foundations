# Git basics I use

## Typical update flow
1. Edit files
2. `git status`
3. `git add .`
4. `git commit -m "Describe what changed"`
5. `git push`

## Commands
- Clone: `git clone <repo_url>`
- Status: `git status`
- Add: `git add .`
- Commit: `git commit -m "message"`
- Push: `git push`
- Pull: `git pull`
